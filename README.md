# dsa-cpp

A hands-on project for my university Data Structures and Algorithms coursework. 

## Project Philosophy

* **Authentic Code:** Every line here is written entirely by me from scratch—no AI generation, no textbook copy-pasting. I build these for the sole purpose of understanding how they work under the hood and satisfy my own curiosity.
* **First Principles & Optimization:** Many of these implementations start with a raw, brute-force approach. They might not always be the most efficient solutions on the first pass, or even on the second. Therefore, do not use my implementations if your goal is 100% guaranteed efficiency.

## Structure
* `/algorithms` — Searching, sorting, and core logic.
* `/data-structures` — Custom implementations and memory practice.
